# pystan_sample
- このプロジェクトはpipenvで管理している
## このリポジトリの目的
- FJCTの課題である時系列データ予測の練習
- 本リポジトリはpystanを使う

## 参考
- Udemy講座【PythonとStanで学ぶ】仕組みが分かるベイズ統計学入門

## ゴール
1. 任意のデータに対して、pystanを用いてデータの予測をすることができる
  - 簡単なモデルの設定
  - モデルのチューニング
2. 予測に伴う手順を説明することができる

## 目的達成へのKPI
- 　pystanの扱い方を学習
  - 単回帰
  - 重回帰
  - ロジスティック回帰
  - 状態空間モデル
- 任意のデータへの適用
- できればブログにまとめる
